# 声明
数据库含有自2010年7月10日至2025年11月15日的每日比特币的一些数据，对这些数据进行分析，由此可以推广到其他更多货币

当中计算时肯定很多是不太准确的，但大体相似。不太准确的原因是上一日的收盘价和本日的开盘价很多不一致，还有就是有些是根据浮点数计算，保留多少位也存在影响

总之，大方向是这样的

注：股市有风险，投资需谨慎

## 数据统计
### 真实数据
列出指定范围的表单数据，还有处理过的日涨幅、最大涨幅、最底涨幅
### 宏观趋势
以周趋势举例，假设过去7天中有4天增长、2天跌、1天不涨不跌，则指数为4-2+0=2
在顺势状态下，如果指数大于0，则会进行购买，将每日实际操作获得的浮动连乘得出结果
### 止盈止损
这个概念不明白就别玩了，不解释了
### 顺势回调
例如顺势涨1%，就是当涨到1%时，则会买入
## 买卖测试

```

卖
卖
```
## 浮动连乘
假设输入以下内容
```
0.3
0.52
-0.9
```
则会进行以下计算
```
(1+0.3)*(1+0.52)*(1-0.9)
```
